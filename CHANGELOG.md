# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-27

### Added
- Initial release
- 10 curated visual style presets (Clean SaaS, Glassmorphic, Brutalist, Cyberpunk, Elegant Serif, Playful Gradients, Dark Mode First, Vintage Retro, Organic Nature, Corporate Professional)
- "From scratch" mode — generate landing pages from product descriptions
- "Improve / convert" mode — enhance existing HTML with preset aesthetics
- Custom aesthetic synthesis — describe your own style, skill blends presets
- Dark/light mode with system preference detection + manual toggle
- Scroll animations via IntersectionObserver + Anime.js
- Responsive design (mobile-first, 375px / 768px / 1280px)
- SEO meta tags (OpenGraph, Twitter Cards)
- Accessibility (semantic HTML, ARIA labels, focus states, skip-to-content)
- FAQ accordion, pricing toggle, animated counters
- One-line install script
- Live demo page (Batteric landing)
- GitHub Pages deployment workflow
